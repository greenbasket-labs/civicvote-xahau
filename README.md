# CivicVote Xahau

CivicVote is a user-facing on-chain voting service built on the Xahau blockchain.

It enforces voting rules directly on-ledger using Xahau Hooks, ensuring transparent, immutable, and trustless voting without relying on centralized servers or administrators.

## Features
- On-chain rule enforcement via Hooks
- One vote per address
- Time-bound voting windows
- Predefined valid options only
- Public and verifiable results

## Use Cases
- Community governance
- DAO signaling
- Public or civic polls
- Educational governance experiments

## Status
Early-stage prototype built for the Xahau Dev Contest (2026).
